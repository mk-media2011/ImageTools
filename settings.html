Output Conversion Options

Image Quality Compression
Adjust the slider to configure the export compression ratio.
  • Applies only to lossy formats: JPEG, WebP, and AVIF.
  • Higher quality (85%-100%): Visually flawless, larger file sizes.
  • Balanced compression (70%-80%): Tiny footprint, highly optimized for online storage.

Filename Prefix Rule
Define a custom string to prepend to every saved image's name.
  • Helps categorize images instantly when doing bulk downloads (e.g., Project_A_).
  • If the original image was named banner.jpg, saving with prefix Work_ results in Work_banner.jpg.

Folder & Interface Settings

Group Under Root Menu
Choose how context actions are displayed.
  • Enabled (Recommended): Groups formats, searches, and sharing options under a clean Image Tools parent menu.
  • Disabled: Displays active formats/searches straight on the main right-click menu for lightning-fast access.
Subfolder Download Saving
Organize downloaded images directly inside a subfolder.
  • Saves folders inside Chrome's main Downloads directory (e.g., Downloads/Saved Images/).
  • Subfolders are created dynamically. Leave blank to save directly into the root Downloads directory.
